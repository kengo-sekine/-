# チェックディジットを求めよ Python編

4 桁の文字列に対して仕様どおりのチェックディジットを計算し、末尾に付加して出力する Python 実装です。

## 問題の整理

- 入力は長さ `n` の文字列リスト `target_list`
- 各要素は長さ 4 の数字文字列
- 各文字列 `N1N2N3N4` について、手順で求めた `N5` を末尾に付けて `N1N2N3N4N5` を出力する

### チェックディジットの手順

1. `N5 = 5*N1 + 4*N2 + 3*N3 + 2*N4`
2. `N5` を `9` で割った余りに置き換える
3. `N5` が `6` の倍数なら `N5 = 1` とし、以降の手順はスキップする
4. `N5` が偶数なら `N5 = N5 // 2`
5. `N5` が `1` なら `N5 = 0`

## ファイル構成

- `app.py`: チェックディジット計算と出力ロジック
- `main.py`: コマンドライン引数をそのまま `app.main()` に渡すエントリポイント
- `tests/test_app.py`: 単体テスト

## 実行方法

```bash
python3 main.py 1234 2345
```

出力:

```text
12343
23454
```

## テスト

```bash
python3 -m unittest discover -s tests -v
```
